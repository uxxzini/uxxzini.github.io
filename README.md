## uxxzini.github.io


### Landscape Architecture × Data × AI Blog

Personal technical blog focused on applying data analysis, AI, and web-based visualization
to urban and environmental problems, grounded in a landscape architecture background.

🔗 **Live site**: https://uxxzini.github.io

---

#### 👋 About This Blog

This blog documents my journey of exploring urban, environmental, and spatial problems
through data analysis, machine learning, and visualization.

With a background in landscape architecture, I focus on:
- Defining problems from a spatial and human-centered perspective
- Analyzing urban and environmental data using Python and AI models
- Communicating insights through clear visualizations and web interfaces

---

#### 📂 Content Structure

- **Projects**  
  End-to-end case studies: problem definition → data → modeling → interpretation

- **Urban & Landscape Data**  
  Landscape architecture competitions, urban theory, and spatial data studies

- **AI Experiments**  
  Machine learning and deep learning experiments, including failures and insights

- **Visualization & Web**  
  Data visualization, dashboards, and UI/UX-oriented web implementations

---

#### 🛠 Tech Stack

- **Languages**: Python
- **Data / AI**: Pandas, NumPy, Scikit-learn, TensorFlow / PyTorch
- **Spatial / GIS**: GeoPandas, QGIS
- **Web**: React
- **Visualization**: Matplotlib, Plotly
- **Blog**: Hugo, PaperMod, GitHub Pages

---

#### 🎯 Purpose

- Build a portfolio demonstrating problem-solving skills at the intersection of
  landscape architecture and data science
- Share practical experiments and insights with others interested in urban data
- Continuously document learning and project development
